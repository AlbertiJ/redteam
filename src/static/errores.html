📚 Códigos HTTP

Cuando la API responde, te tira un número. Acá te explico qué significa cada uno.

200 OK

Todo bien. La request funcionó.

400 Bad Request

Tu request está mal. Te falta un campo, o mandaste un tipo de dato incorrecto.

401 Unauthorized

Necesitás estar autenticado. Te falta el token o el token es inválido.

403 Forbidden

Estás autenticado pero no tenés permiso para este recurso.

404 Not Found

El recurso no existe. La URL está mal, o el ID no corresponde a nada.

405 Method Not Allowed

El verbo HTTP no está permitido para este endpoint. Ej: mandás POST a algo que solo acepta GET.

500 Internal Server Error

El server explotó. Bug. Reportalo si lo ves.

503 Service Unavailable

El server está sobrecargado o en mantenimiento.
