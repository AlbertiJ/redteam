🆕 Crear DB Custom

💡 Recomendaciones al diseñar tu DB:
• Si vas a guardar emails: respetá el formato (usuario@dominio.com).
• Si vas a guardar teléfonos: incluí código de área (+54 11, +54 221, etc.).
• Si vas a guardar IDs: usá valores "lejanos" (1000, 1001...) para que se note el patrón en la enumeración.
• El primer campo debe llamarse "id" (convención del lab).
• Mín 3 campos, máx 100.
