🗄️ Cargá tu DB en 1 paso

Pegá un JSON con tu DB y los datos. La DB se crea sola, los datos se cargan, listo.

Formato: {"nombre": "...", "campos": [...], "datos": [...]}

Ejemplo abajo. Si querés algo más simple, en Práctica hay 7 DBs ya listas.
↓ Ejemplo — copialo y modificalo:
{ "nombre": "bandas_favoritas", "campos": ["id", "banda", "tema", "anio"], "datos": [ {"id":"1", "banda":"Soda Stereo", "tema":"De música ligera", "anio":"1990"}, {"id":"2", "banda":"Patricio Rey", "tema":"Calavera", "anio":"1985"}, {"id":"3", "banda":"Redondos", "tema":"Jijiji", "anio":"1986"} ] }

¿Te trabaste?

Si el JSON te confunde, probá primero con las 7 DBs precargadas y volvé acá después.
